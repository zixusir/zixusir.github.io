第6章 构建属于自己的定投计划

理财 2019/05/12

一份完整的，好的定投计划需要根据自己的收入和开支，构建合理的每月定投额度。需要筛选合适投资的品种，设定好买卖区间。最终的计划必须能够落在纸面上，这样才能在每次投资的时候严格遵守，避免主观情绪的影响。

1. 梳理自己的现金流

股票基金类的资产，需要做好3-5年不会使用的准备。通过记账，避免不必要的消费。刨除必要生活开支后，剩余自己的50%可以用于定投。

2. 挑选好基金

每次投资前，查询当前适合投资的基金有哪些。通过银行螺丝钉公众号来查看。刚开始建议从红利、宽基等指数基金开始。

3. 构建定投计划

选择定投渠道：场内/场外
定投频率：按周，按月，刚发下工资的第二天

4. 定期检查优化

在定投的同时，做好记录，方便总结回顾，记录下自己买入的日期、品种、操作、价格、买入份额、估值
范例：
|日期|操作（买/卖）|交易品种代码|买入或卖出的金额|成交单价|买入或卖出的份额|买入或卖出时的估值|

5. 为父母构建养老定投计划

当父母退休后，他们可以从计划中定期取用资金。
资金需要很长的时间只进不出，甚至包括达到卖出条件以后，卖出的资金也会被继续用于投资。需要等待收益提升，直至覆盖所需的生活开支。
一般来说，养老定投计划需要高分红的指数基金，比如红利指数基金，这是因为红利指数基金一般高分红，依靠指数基金每年的现金分红，就可以稳定的获取现金。
如何做：

  1. 挑选高股息的指数基金，例如红利指数基金
  2. 根据自己的需求，决定定投的年限
  3. 在指数基金处于低估的时候，开始我们的定投计划。
  4. 遵循本课程的改进定投策略

6. 为自己构建加薪定投计划

对于年轻的上班族来说，认真工作才是让自己资产变多的最佳途径，而不是靠投资

上班族投资的三原则：

  1. 钱少也要存，节流为先
  2. 区分“想要”和必要
  3. 投资越早开始越好

如何做：

  1. 把月收入结余的60%用于定投指数基金
  2. 挑选当前处于低估的指数基金
  3. 开始我们的定投计划
  4. 遵循本课程改进的定投策略

7. 为子女构建教育定投计划

特点：资金的使用时间和金额有很强的可预见性

教育定投的原则：

  1. 保证资金需要用的时间可以拿来用
  2. 其次才是提高收益
  3. 卖出条件要放宽，临近使用时间要转为保本理财产品打理

卖出条件放宽指的是可以在指数基金进入正常就可以开始卖出

如何做：

  1. 根据孩子上学需要用钱的时间，确定定投的年限
  2. 挑选当前处于低估的指数基金
  3. 开始定投
  4. 遵循本课程的改进定投策略，同时放宽卖出条件
